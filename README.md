# 🛠️ Scrollforge API

**Scrollforge API** is a handcrafted, lore-rich fantasy character generator inspired by the worldbuilding and themes of The Elder Scrolls universe.  
It creates detailed, original character profiles based on races, classes, regions, factions, and more — perfect for fans, modders, writers, and D&D players.

> 🔮 "Born from the ash of Vvardenfell and the blood of forgotten kings, Scrollforge creates legends one line of JSON at a time."

---

## 🚀 Features

- Generate randomized fantasy character profiles
- Richly detailed with race, class, location, faction, backstory, and more
- Built with Flask and fully JSON-based
- Safe, copyright-free lore emulation
- Customizable data via modular `.json` files

---

## 🎯 Example Output

json
{
  "name": "Vael Seron",
  "race": "Dunmer",
  "class": "Assassin",
  "region": "Vvardenfell",
  "faction": "Morag Tong",
  "birthsign": "The Shadow",
  "alignment": "Neutral Evil",
  "backstory": "Vael was born in the ash-choked alleys of Balmora..."
}```

📡 API Endpoint
GET /generate
Returns a fully-formed character in JSON.

Optional query parameters (coming soon):
?race=nord

?region=skyrim

?faction=companions

🧱 Tech Stack
Backend: Python + Flask
Data Source: JSON → SQLite (future))
API Publishing: RapidAPI (planned)
Premium Features: AI-enhanced bios (future)
Optional Frontend: Next.js / Flask templates
