# Scrollforge API

**Scrollforge API** is a lore-inspired fantasy character generator based on Elder Scrolls-style worldbuilding.

## Current Status
🛠️ MVP under construction!

## Usage
Make a GET request to `/generate` to receive a fantasy character profile.

## Example
```json
{
  "name": "Vael Seron",
  "race": "Dunmer",
  ...
}
